# Analisis Sentimen Kesehatan Mental di Twitter (X)

## 📊 Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis sentimen tweet tentang kesehatan mental menggunakan tiga algoritma machine learning:

- **Naive Bayes**
- **K-Nearest Neighbor (KNN)**
- **Random Forest**

Dashboard interaktif ini memungkinkan pengguna untuk:
- Melihat perbandingan performa ketiga model
- Melakukan prediksi sentimen real-time
- Memahami proses preprocessing teks
- Melihat hasil cross validation
